## Dependencies
- Vulkan SDK (graphics API)
- GLFW (window library)
- GLM (math library)
- GLSLC (shader compiler)
  (C\+\+17, g\+\+)

# 

Everything **except the Vulkan SDK** is included here

You need to install the **Vulkan SDK** from [**here**](https://vulkan.lunarg.com), if you have not already
