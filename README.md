## Dependencies
#### Need to install
- [Vulkan SDK](https://vulkan.lunarg.com "https://vulkan.lunarg.com")
#### Provided in [libs](https://github.com/vrecusko/Dasoku/tree/master/libs "https://github.com/vrecusko/Dasoku/tree/master/libs")
- [GLFW](https://github.com/glfw/glfw "https://github.com/glfw/glfw")
- [GLM](https://github.com/g-truc/glm "https://github.com/g-truc/glm")
- [GLSLC](https://github.com/KhronosGroup/glslang "https://github.com/KhronosGroup/glslang")
- [ImGui](https://github.com/ocornut/imgui "https://github.com/ocornut/imgui")
- [STB](https://github.com/nothings/stb "https://github.com/nothings/stb")
- [tinyobjloader](https://github.com/tinyobjloader/tinyobjloader "https://github.com/tinyobjloader/tinyobjloader")
